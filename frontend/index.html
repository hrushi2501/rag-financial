Help & Troubleshooting

Quick start

1. Upload

Drop PDF/DOCX/TXT/CSV files in Upload. OCR for images in DOCX is supported when enabled.

2. Search

Use natural language queries. Tune results count and threshold for recall/precision.

3. Chat

Ask questions with “Include Context” to ground answers. Citations appear as chips.

Health states

Online Degraded Offline
  • Online: All services are healthy.
  • Degraded: External services (Pinecone/Embeddings) are slow or partially unavailable; retries may occur.
  • Offline: Backend unreachable or critical failure. Check your connection and server logs.

Troubleshooting

Upload issues

  • Max size ~10MB per file.
  • Supported: PDF, DOCX, TXT, CSV.
  • Scanned PDFs need external OCR (e.g., Google Vision) or pre-processing.

Search quality

  • Lower threshold for more recall; raise for more precision.
  • Increase Results for broader context.

Connectivity

  • If offline, ensure internet is connected.
  • API 503/429 indicates overload or rate limiting; try again shortly.
